# 🧬 OncoConnect Co-Creation Hub

AI-supported proposal development and partner co-creation platform for the
OncoConnect Horizon Europe project.

## Features

- 🔐 Role-based authentication (Admin / Partner / Patient)
- 📊 Interactive dashboard with countdown timer
- 📦 Work package management & Gantt chart
- 💬 Partner & patient feedback collection
- 🗳️ 3-country approval gate with download control
- 📢 Announcement board
- 📁 Document management
- 🛡️ Admin panel with state management

## Roles

| Role | Access |
|------|--------|
| Admin | Full access + download + approve override |
| Partner | Feedback + approve own country |
| Patient | Feedback only |
